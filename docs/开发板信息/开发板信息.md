## emmc信息

```shell
=> mmc list
dwmmc@fe320000: 1
sdhci@fe330000: 0 (eMMC)
```

```shell
+	aliases {
+		mmc0 = &sdio0;
+		mmc1 = &sdmmc;
+		mmc2 = &sdhci;
+	};
```

这里配置会导致磁盘盘符为```/dev/mmc2blkp*```
